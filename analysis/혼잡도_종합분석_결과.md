# 시간대별 지하철 혼잡도 종합 연관성 분석 보고서

## 분석 개요

### 분석 목적
서울 지하철 혼잡도에 영향을 미치는 다양한 도시 요인들을 종합적으로 분석하여, 혼잡도 결정 요인의 상대적 중요도와 시간대별 패턴을 규명한다.

### 분석 대상
| 항목 | 내용 |
|------|------|
| 분석 노선 | 서울 지하철 1~8호선 (종합), 2·4·5호선 (심층) |
| 분석 역 수 | 약 180개 역 |
| 분석 단위 | 역별 / 행정동별 |
| 기준 데이터 | 평일 시간대별 혼잡도 |

### 분석 요인
| 요인 | 데이터 출처 | 분석 내용 |
|------|-------------|-----------|
| 직장인구 | 행정동별 직장인구 통계 | 통근 수요와의 관계 |
| 유동인구 | 행정동별 시간대별 유동인구 | 상권 활동과의 관계 |
| 추정매출 | 행정동별 업종별 매출 | 상업 활동과의 관계 |
| 역세권 건물 | 역 반경 내 건물 특성 | 토지이용과의 관계 |

## 3. 요인별 상세 분석 결과

### 3.1 직장인구와 혼잡도

| 지표 | 값 | 해석 |
|------|-----|------|
| 전체 상관계수 | 0.1095 | 약한 양의 상관관계 |
| 출근시간대 | 0.0473 (p=0.21) | **유의하지 않음** |
| 퇴근시간대 | **0.1616** (p<0.001) | **유의함** |
| 최고 상관시간 | 17:30 (r=0.317) | 퇴근 직전 |
| 상/하위 25% 차이 | **4.79%p** | 유의미한 차이 |

**주요 발견**:
- 출근 시간: 직장인구와 혼잡도 간 유의미한 관계 없음
- 퇴근 시간: 직장인구 밀집 지역에서 승차 집중으로 혼잡도 상승
- 17:30에 최고 상관계수 기록 (퇴근 직전 피크)

### 3.2 유동인구와 혼잡도

| 지표 | 값 | 해석 |
|------|-----|------|
| 전체 상관계수 | 0.0745 | 약한 양의 상관관계 |
| 시간대 매칭 분석 | **0.1516** | 2배 증가 |
| 출근시간대 | 0.0754 (p=0.04) | 약하게 유의 |
| 퇴근시간대 | 0.0846 (p=0.02) | 약하게 유의 |
| 최고 상관시간 | **21:30** (r=0.181) | 야간 |
| 상/하위 25% 차이 | **3.73%p** | 유의미한 차이 |

**주요 발견**:
- 직장인구보다 낮은 상관관계 (0.075 vs 0.110)
- 야간 시간대(20:00~22:00)에 가장 높은 상관관계
- 출퇴근 피크 시간에는 오히려 상관관계 약함

## 6. 종합 결론

### 6.1 핵심 발견 사항

#### 1. 혼잡도 결정 요인 순위
```
1위: 역세권 건물 특성 (업무건물 r=0.35)
2위: 추정매출 (r=0.20~0.35)
3위: 직장인구 (r=0.11)
4위: 유동인구 (r=0.07)
```

#### 2. 시간대별 주요 영향 요인
| 시간대 | 주요 요인 | 설명 |
|--------|-----------|------|
| **출근** | 주거건물, 총연면적 | 주거지역에서 출발하는 통근 수요 |
| **점심** | 업무건물, 상업건물 | 점심식사 이동 수요 |
| **퇴근** | 업무건물, 직장인구 | 업무지역에서 귀가하는 통근 수요 |
| **야간** | 업무건물, 유동인구 | 퇴근 후 상업/문화 활동 |

#### 3. 직장인구 vs 유동인구
| 비교 항목 | 직장인구 | 유동인구 |
|-----------|----------|----------|
| 전체 상관계수 | **0.110** | 0.075 |
| 최고 상관 시간대 | 퇴근 (17:30) | 야간 (21:30) |
| 상/하위 혼잡도 차이 | **4.79%p** | 3.73%p |
| 주요 영향 | 통근 수요 | 여가/소비 활동 |

> **결론**: 지하철 혼잡도는 유동인구보다 **직장인구**와 더 강한 관련성을 보이며, 이는 지하철이 주로 **통근 목적**으로 이용되기 때문

### 6.2 정책적 시사점

| 영역 | 시사점 |
|------|--------|
| **혼잡 완화** | 업무 밀집 지역 퇴근 시간대 집중 관리 필요 |
| **배차 계획** | 주거지역 출근, 업무지역 퇴근 시간대 증차 |
| **도시 계획** | 주거-업무 혼합 개발로 양방향 혼잡도 분산 |
| **상권 분석** | 혼잡도 단독보다 복합 지표 활용 권장 |

### 6.3 분석 한계 및 향후 과제

**분석 한계**:
- 단일 요인 설명력 한계 (R² < 0.4)
- 환승역 복잡성 미반영
- 계절/이벤트 변동 미분석

**향후 과제**:
- 전 노선 확대 분석
- 실시간 데이터 기반 예측 모델
- 딥러닝 기반 혼잡도 예측
- 버스/자전거 연계 분석
