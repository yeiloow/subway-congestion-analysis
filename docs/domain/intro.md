# 사전 설명

지하철은 관리 주체가 다양함.

1. 서울교통공사
2. 코레일
3. 민간

지하철 관리 주체가 다른만큼 데이터도 별도로 제공되는 경우가 많음.
노선 구별 없이 동일한 기준으로 구할 수 있는 데이터도 있지만 아닌 데이터도 많음.
결측치 최소화를 위해 데이터가 제일 많이 제공되는 서울교통공사 역들을 위주로 데이터를 정리함.
혼잡도 계산 정보를 구하기 힘든 1~9호선 라인 이외에 노선은 제외함

## 역번호

지하철에는 역마다 고유한 역번호가 존재함.
우리가 지하철에서 볼 수 있는 번호는 외부 번호로 고객에게 안내를 위한 번호임.
이런 번호는 설명을 위해 사용되며 까치산역 "234-4"은 신도림역 234에서 갈라져 나온 4번째역으로 읽을 수 있음.
하지만 지하철 내부에선 "-" 같은 특수기호를 최소화하기 위해 특수기호가 없는 내부번호가 별도로 존재함.
본프로젝트도 마찬가지로 분석에 있어서는 데이터 정제를 위해 내부번호를 사용함.

## 역 선정 기준

데이터 훈련을 위해 2023년부터 존재했던 역들만 추출함.
